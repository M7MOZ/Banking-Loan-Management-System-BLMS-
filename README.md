## Project Overview
This repository contains database testing artifacts for a Banking Loan Management System (BLMS).  
The project focuses on validating database integrity, business rules, and relational constraints using SQL queries.

Testing was performed at the **database level only**, without any UI or API interaction.

## Scope of Testing
Modules tested:

- Customer Management
- Account Management
- Loan Management
- Collateral Management
- Transaction Management
- Fees and Penalties
- Branch and Employee Management

Testing focused on validating:
- Constraints and ENUM values
- Foreign key relationships
- Business rules
- Trigger-based calculations

## Database
Tables included in the system:

- Customers
- Accounts
- Loans
- Collaterals
- Transactions
- Fees
- Branches
- Employees

## Testing Approach
Testing was performed using:

- INSERT queries to create test data
- UPDATE queries to simulate system behavior
- SELECT queries to validate results
